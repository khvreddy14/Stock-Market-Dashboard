# Stock-Market-Dashboard
Stock Market Dashboard using Python  Developed an interactive stock market dashboard using Python, Streamlit, and yFinance.  Implemented real-time stock data retrieval and visualization.  Displayed key financial metrics such as current price, market capitalization, and yearly highs.  Created dynamic charts and tabular reports for stock performance.

# Stock Market Dashboard

A Python-based stock market dashboard that displays real-time stock information and historical trends.

## Features

- Search stocks by ticker symbol
- Real-time stock information
- Market capitalization
- 52-week high
- Historical stock chart
- Interactive dashboard

## Technologies

- Python
- Streamlit
- yFinance
- Pandas

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run main.py
```
